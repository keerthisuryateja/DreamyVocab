# Usage Guide

## Searching for a Word

1. Type a word in the search box at the top.
2. Choose a **source** from the dropdown:
   - **Auto** — tries the best available source automatically
   - **Merriam-Webster** — authoritative English dictionary
   - **Wiktionary** — broader coverage including slang and technical terms
3. Press **Enter** or click **Search**.

The definition card appears with the word, its meaning, and which source it came from.

---

## Saving a Word

After a successful search, click **Save Word** on the definition card.  
The word is added to your local database. The **Today** and **Total** counters in the header update instantly.

If the word is already saved, the button label changes to reflect that.

---

## Viewing Suggestions

As you type, a suggestion list may appear below the search box showing related words. Click any suggestion to search it directly.

---

## Zoom / Accessibility

| Shortcut | Action |
|----------|--------|
| `Ctrl` + `+` | Zoom in |
| `Ctrl` + `-` | Zoom out |
| `Ctrl` + `0` | Reset zoom |

---

## Your Word Database

All saved words are stored locally in `%APPDATA%\PersonalVocabTracker\vocabulary.db`.  
See [Installation.md](Installation.md) for backup and migration tips.
